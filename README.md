# HamOps — de KJ5MJD

Ham radio operator utilities for macOS, iPhone, and iPad.

## Download for Mac

[⬇ Download HamOps v1.0 for macOS](https://github.com/rhenton75035/HamOps/releases/latest)

Requires macOS 13 (Ventura) or later.

> **First launch:** Right-click HamOps.app → Open → click Open 
> to bypass the unsigned app warning. Only needed once.

## Features

- Real-time solar conditions — SFI, K-index, A-index, sunspots
- MUF, OWF, and foF2 from live ionosonde data
- HF band propagation forecast 160m–6m
- Noise floor estimates — 80m / 40m / 20m / 15m / 10m
- Live DX cluster spots with mode filtering
- North American band plan — FCC Part 97 and ISED RBR-4
- Maidenhead grid locator with GPS
- Frequency and antenna calculator with SWR
- FCC callsign lookup
